Конструирование и сохранение локаций для 2D иргы на Android (не все элементы добавлены)

Все элементы могут располагаться либо на нижней части экрана, либо крепится к "препятсвию" (обозначается как Obstacle на всплывающих подсказках)
Сохранение происходит в xml файл, который в последствии подгружается в Android проект
